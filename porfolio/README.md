## How to Use

1. Open `home.html` in a web browser
2. Navigate through the pages using the navigation menu
3. All links and scripts should work properly

## Customization

### Update Personal Information
- Replace "Your Name" with your actual name in `home.html`
- Update email and contact information in `contact.html`
- Add your own biography and skills in `about.html`

### Add Your Projects
- Edit `projects.html` to add your own projects
- Update project images (currently using placeholders)
- Add your GitHub and live demo links

### Styling
- Modify colors in `styles.css` (search for color codes like `#667eea`)
- Adjust spacing, fonts, and animations as needed

### Images
- Replace placeholder images with your own
- Recommended image sizes:
  - Profile picture: 500x500px (square)
  - Project screenshots: 400x250px
